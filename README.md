
# Customer Data Cleaning Project

## Overview
Cleaned a large customer dataset (~99k records) using Python and Pandas.

## Tasks Performed
- Removed duplicate records
- Handled missing values
- Standardized city and state names
- Validated zip codes
- Exported cleaned dataset

## Tools Used
- Python
- Pandas
- Google Colab

## Outcome
Prepared a structured dataset ready for further analysis and visualization.
